# TRAVIR: 메타버스 기반 관광지 추천 서비스

<img src="https://github.com/kim0104/Travir/assets/81177557/f0ae5acd-5bba-4223-8f0d-87497a6e3638" width="700" alt="TRAVIR 메인 이미지">

## 📜 서비스 소개
2021년을 기점으로, 메타버스 시장은 전례 없는 속도로 성장하며 다양한 산업에 혁신적인 변화를 가져왔습니다. 이러한 변화는 특히 여행 산업에서 두드러지게 나타났는데, 전 세계적인 여행 제약과 변화하는 여행 수요에 대응하여, 메타버스를 통한 새로운 관광 경험 제공이 주목받기 시작했습니다. 특히, 여러 이유로 여행이 어려운 이들을 위해 메타버스가 제공하는 가상 여행은 실제와 가까운 경험을 제공함으로써 새로운 가능성을 열었습니다.  
TRAVIR는 이러한 시대적 변화에 발맞춰, 사용자에게 가상 공간에서의 관광 체험과 맞춤형 관광지 추천을 제공하는 서비스입니다.


### 목표
- 서울과 제주도를 상징하는 가상공간을 구현하여 관광지를 체험하는 새로운 방법을 제시
- 기존에 사용자의 번거로운 여행지 검색 과정을 줄여주고 간단한 여행 키워드 선택을 통해 맞춤 여행지를 추천하여 원하는 여행지를 쉽고 빠르게 결정
